# RadioChipGroup

RadioChipGroup allows a user to select a single item from a list of mutually exclusive options.

components:

- **`<RadioChipGroup>`**
- **`<RadioChipGroupSection>`**
- **`<RadioChip>`**

In forms use the **`<RadioChipGroupField>`** component (see [Forms > Fields > RadioChipGroupField](/docs/forms-fields-RadioChipGroupfield--docs))

### Usage

<Source

<RadioChipGroup
  name={name} {/* string - the fieldset name */}
  legend={legend} {/* string - the fieldset label/legend - this SR only */}
  defaultValue={defaultValue} {/* string, optional - default value of the group */}
  srOnlyLegend={srOnlyLegend} {/* boolean - visually hides the group legend */}

  disabled={disabled} {/* boolean - disables the whole field */}
>
  <RadioChipGroupSection

  >
    <RadioChip
      value={value} {/* string - the radio item value */}
      label={label} {/* string - the radio item label */}
      iconName={iconName} {/* optional - type of Icon names (see Icon docs for names) */}
      disabled={disabled} {/* boolean - disables this Chip item only */}
    />
    { ...more items }
  </RadioChipGroupSection>
 { ...more sections }
</RadioChipGroup>
`}

### Props notes

**`<RadioChipGroup>`**

- `legend` is a screen-reader-only label for the fieldset (added as the `aria-label` attribute)
