### Props notes

**`<RadioCardGroup>`**

- `orientation` : `'vertical' | 'horizontal'` - default is `'vertical'`

In horizontal orientation wrap each item in a `<div>`, so it does not take the full width of the group.

---

**`<RadioCard>`**

- `rightSlot` : optionally, a right-side element can rendered on the right side of the card.
- `bottomSlot` : optionally, a bottom-side element can rendered on the bottom side of the card.

The slots must not contain any interactive elements or other inputs, they are for static elements only such as displaying extra information or images.
