# DatePicker

**`<DatePicker>`** combines a Date input and a Calendar popover to allow users to enter or select a date value.

In forms use the **`<DatePicker>`** component (see [Forms > Fields > DatePicker](/docs/forms-fields-datepickerfield--docs)).
