# Textarea

**`<Textarea>`** is a multiline text input, useful for cases where users have a sizable amount of text to enter.

In forms use the **`<TextareaField>`** component (see [Forms > Fields > TextareaField](/docs/forms-fields-textareafield--docs)).

<Canvas of={TextareaStories.Default} />

### Usage

<Source code={`import { Textarea } from '@acme/ui'`} />

<Source
	code={`
<Textarea
  label={label} {/* string */}
  {/* optional props */}
  id={id} {/* string */}

  srOnlyLabel={srOnlyLabel} {/* boolean */}
  error={error} {/* boolean */}

  readOnly={readOnly} {/* boolean */}
  onChange={(event: Event) => {}} {/* function, change event listener */}
  required={required} {/* boolean - "(optional)" shown on label if !required & showOptional={true} */}
