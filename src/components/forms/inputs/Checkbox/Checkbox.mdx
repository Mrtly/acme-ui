# Checkbox

**`<Checkbox>`** is used to mark one individual item as selected (or not).

Checkboxes can also allow the user to select multiple items from a list of individual items.

This is an individual item, suitable for a `boolean` value input.

For a group of Checkboxes use the [CheckboxGroupField](/docs/inputs-checkboxgroup--docs).

In forms use the **`<CheckboxField>`** component (see [Forms > Fields > CheckboxField](/docs/forms-fields-checkboxfield--docs)).
