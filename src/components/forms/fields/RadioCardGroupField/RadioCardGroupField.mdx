# RadioCardGroupField

A **`<RadioCardGroupField>`** component to be used in Forms

For non-form purposes, utilize the base **`<RadioCardGroup>`** input. [Forms > Input > RadioCardGroup](/docs/inputs-radiocardGroup--docs).

<Canvas of={RadioCardGroupFieldStories.Default} />

### Usage

Use **`<RadioCardGroupField>`** inside the **`<Form>`** wrapper component (see [Form > Docs](/docs/forms-fields-form--docs))

<Source code={`import { RadioCardGroupField, RadioCardGroupFieldItem } from '@acme/ui'`} />

<Source
	code={`
<RadioCardGroupField
  control={form.control} {/* connect to form control */}

### Props notes

**`<RadioCardGroupField>`**

- `orientation` : `'vertical' | 'horizontal'` - default is `'vertical'`

In horizontal orientation wrap each item in a `<div>`, so it does not take the full width of the group.

---

**`<RadioCardGroupFieldItem>`**

- `rightSlot` : optionally, a right-side element can rendered on the right side of the card.
- `bottomSlot` : optionally, a bottom-side element can rendered on the bottom side of the card.

The slots must not contain any interactive elements or other fields, they are for static elements only such as displaying extra information or images.
