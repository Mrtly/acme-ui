# RadioGroupField

A **`RadioGroupField`** component to be used in Forms

For non-form purposes, utilize the base **`<RadioGroup>`** input. [Forms > Input > RadioGroup](/docs/inputs-radiogroup--docs).

<Canvas of={RadioGroupFieldStories.Default} />

### Usage

Use **`<RadioGroupField>`** inside the **`<Form>`** wrapper component (see [Form > Docs](/docs/forms-fields-form--docs))

<Source code={`import { RadioGroupField, RadioGroupFieldItem } from '@acme/ui'`} />

<Source
	code={`
<RadioGroupField
  control={form.control} {/* connect to form control */}
  name={name} {/* string */}
  legend={legend} {/* string */}
  description={description} {/* string, recommended - add a description for better accessibility */}
  {/* optional props */}
  srOnlyDescription={srOnlyDescription} {/* boolean - visually hides the description */}
  srOnlyLegend={srOnlyLegend} {/* boolean - visually hides the legend */}
  disabled={disabled} {/* boolean */}
  readOnly={readOnly} {/* boolean */}
  required={required} {/* boolean, default true - "(optional)" shown on label if !required  */}
  showOptional={showOptional} {/* boolean - can be set to false for inputs not part of a form ie. Searchbar */}
  orientation="vertical" {/* "vertical" || "horizontal" */}
>
  <RadioGroupFieldItem 
    label={label} {/* string */}
    value={value} {/* string */}
    disabled={disabled}} {/* boolean */}
  />
  {/* ...more items */}
</RadioGroupField>
`}
/>

### Props notes

**`<RadioGroupField>`**

- `className` : additional classes can be applied to the fieldset
- `orientation` : `'vertical' | 'horizontal'` - default is `'vertical'`

In horizontal orientation wrap each item in a `<div>`, so it does not take the full width of the group.
