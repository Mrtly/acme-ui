# CheckboxCardField

A **`<CheckboxCardField>`** component to be used in Forms.

This is an individual form item, suitable for a `boolean` value form field.

For a group of Checkboxes or CheckboxCards use the [CheckboxGroupField](/docs/forms-fields-checkboxgroupfield--docs).

For non-form purposes, utilize the base **`<CheckboxCard>`** input. [Forms > Input > CheckboxCard](/docs/inputs-checkboxcard--docs).

<Canvas of={CheckboxCardFieldStories.Default} />

### Usage

Use **`<CheckboxCardField>`** inside the **`<Form>`** wrapper component (see [Form > Docs](/docs/forms-fields-form--docs))

<Source code={`import { CheckboxCardField } from '@acme/ui'`} />

<Source
	code={`
<CheckboxCardField
  control={form.control} {/* connect to form control */}
  name={name} {/* string */}
  label={label} {/* string */}
  {/* optional props */}
  description={description} {/* string - recommended, add a description for better accessibility */}
  id={id} {/* string */}
  srOnlyDescription={srOnlyDescription} {/* boolean */}
  disabled={disabled} {/* boolean */}
  readOnly={readOnly} {/* boolean */}
  required={required} {/* boolean */}
  showOptional={showOptional} {/* boolean - "(optional)" shown on label if !required  */}
  defaultChecked={defaultChecked} {/* boolean - sets a default value */}
  rightSlot={rightSlot} {/* ReactNode - right side content */}
/>
`}
/>

### Props notes

Optionally, a `rightSlot` prop can be passed to render additional content on the right side of the card.
