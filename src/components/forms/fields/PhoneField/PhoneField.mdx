# PhoneField

A **`<PhoneField>`** component to be used in Forms.

It only accepts digits, and it masks the phone value to the `(000) 000-0000` pattern.

<Canvas of={PhoneFieldStories.Default} />

### Usage

Use **`<PhoneField>`** inside the **`<Form>`** wrapper component (see [Form > Docs](/docs/forms-fields-form--docs))

Use the `PhoneFieldSchema` or see the [Zod schema docs](/docs/forms-zodschemas--docs#predefined-schemas) for a schema breakdown.

<Source
	code={`import { PhoneField, PhoneFieldSchema, Form } from '@acme/ui'
import { useForm } from 'react-hook-form'
import { zodResolver } from '@hookform/resolvers/zod'
import * as z from 'zod'
`}
/>
