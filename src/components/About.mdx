### @acme/ui - a component library

<hr />

A collection of samples of my work from the last couple years.

- various **custom UI components** based on **radix** primitives ([Radix UI](https://www.radix-ui.com/))
- a **[forms system](/docs/forms-form--docs)** with **inputs** & **field** wrappers using **react-hook-form** and **zod** (some based on adobe's [**react-aria-components**](https://react-spectrum.adobe.com/react-aria/components.html))
- a **[data table](/docs/display-datatable--docs)** system wrapping [**tanstack table**](https://tanstack.com/table/latest)
- **[alert](/docs/notifications-alert--docs)** & **[toast/snackbar](/docs/notifications-toast--docs)** notifications systems inlc. components & state/context hooks
- an [**`<Icon>`** wrapper component](/story/theme-icons--all-icons) using the **react-feather** icons library
- more!
