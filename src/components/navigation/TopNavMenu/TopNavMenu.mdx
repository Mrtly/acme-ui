# TopNavMenu

A navigation component that can be used to show a list of navigation links positioned in the top/header of a page or application.

Renders a `<nav>` element with links and lists of links.

<Canvas of={TopNavMenuStories.Default} />

### Usage

**`<TopNavMenuListItem>`** renders a (`<li>`) element.
Its direct children must be either **`<SideNavMenuSection>`** or **`<SideNavMenuDetailsSection>`**

**`<TopNavMenuButton>`** renders a (`<button>`) element, used for calling an action. The `onClick` property is required.

<Source
	code={` import { TopNavMenu, TopNavMenuListItem, TopNavMenuGroup, TopNavMenuButton } from '@acme/ui'`}

### Props notes

- `className` : additional classes can be applied to the TopNavMenu components

**`<TopNavMenu>`**

- `ariaLabel` : `string`, optional, the default aria-label is 'Top Navigation Bar'

---

**`<TopNavMenuGroup>`**

- `title` : `string`, required
- `hasCurrentChildItem` : `boolean`, used for accessibility & styling, optional

---

**`<TopNavMenuListItem>`**

- `href` : `string`, required
- `isCurrent` : `boolean`, used for accessibility & styling, optional
- `icon` : slot prop for an Icon component, optional

---

**`<TopNavMenuButton>`**

- `onClick` : function
- `icon` : slot prop for an Icon component, optional
