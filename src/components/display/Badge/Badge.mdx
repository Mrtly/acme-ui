# Badge

**`<Badge>`** is a small non-interactive element that displays status information for UI elements.

<Canvas of={BadgeStories.Default} />

### Usage

A **`<Badge>`** is used to highlight a UI elemnent's status for quick recognition.

The correct choice of color `variant` is important as it also conveys a message.

<Source code={`import { Badge } from '@acme/ui'`} />

<Source code={`<Badge variant={variant}> { label } </Badge>`} />

### Props notes

- `variant` : `'default' | 'secondary' | 'outline' | 'info' | 'success' | 'warning' | 'error'`
- `className` : additional classes can be applied to the Badge component
